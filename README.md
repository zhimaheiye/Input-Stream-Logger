# Input-Stream-Logger
基于 Rime 输入法（Lua）和 CapsWriter-Offline 的本地输入流记录器。无感捕获每日打字与语音输出，自动保存为日志，用于构建个人 AI 知识库的原始数据集。 A passive input stream logger integrating Rime (Lua) and CapsWriter-Offline. Captures your daily typing and voice outputs to build a personalized AI knowledge base (Second Brain) data source.
